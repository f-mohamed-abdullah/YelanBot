# Installation
- Copy this command and run it in git bash/terminal.
  ```bash
  > git clone https://github.com/mdxabu/YelanBot.git
  ```
- Change the directory to the project.
  ```bash
  > cd YelanBot
  ```
- Run it on your favorite IDE (Intellij Idea/Eclipse)
